## Quick start
To switch between different version of nodesjs. You can use the utility:
https://github.com/nvm-sh/nvm
```bash
$ nvm use
$ npm install
```
Create `.env` file. For example `.env.template`. You can take test .env values from project vault

## Running the app

```bash
# watch mode
$ npm run migragtions

# watch mode
$ npm run dev

# production mode
$ npm run build
$ npm run start
```
